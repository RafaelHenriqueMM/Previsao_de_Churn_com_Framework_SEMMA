## Previsão de Churn com Framework SEMMA
Projeto da disciplina "Aprendizado de Máquina" cujo o objetivo é desenvolver um modelo preditivo de Machine Learning capaz de
identificar usuários com alta probabilidade de "Churn" (evasão/cancelamento), aplicando rigorosamente as etapas do framework analítico SEMMA, seguindo o passo a passo do guia fornecido.


## Instalação
```bash
 # Clone o repositório
git clone https://github.com/RafaelHenriqueMM/Previsao_de_Churn_com_Framework_SEMMA.git

# Crie o ambiente virtual
python -m venv venv

# Ative o ambiente virtual

# Windows:
venv\Scripts\activate

# Linux/macOS:
source venv/bin/activate

# Instale as dependências
pip install -r requirements.txt

```
